- TUI picker now matches widget initialisms in addition to substrings: typing `gb` finds `git-branch`, `tt` finds `tokens-total`, `su` finds `session-usage`. The implementation splits widget `type` and display `name` on `-`, `_`, and whitespace, then prefix-matches the first letter of each token. Single-letter queries stay substring-only so they don't blow up the result set. Substring matching is preserved as a fallback so existing muscle memory (typing `branch`, `model`, `skills`) keeps working.

- New `key-hints` widget (custom group): renders a Claude Code REPL keyboard-shortcut hint that rotates over time so the line keeps reminding the user about a different binding every interval. Per the determinism rule (§1.2 N7) the rotation is seeded from `ctx.clock.now()` — `idx = floor(now / intervalMs) % hints.length` — so identical clock + config still produce byte-stable output and the golden harness stays untouched. Options: `intervalMs` (default 30 s, clamped to ≥ 1 s so a hostile config can't rotate per render), optional `hints: string[]` to override the catalogue (blanks dropped; an all-blank list falls back to the built-in set), and the standard `label`. Built-in catalogue covers ten common shortcuts: `/ commands`, `@ file`, `! bash`, `# memorize`, `Esc interrupt`, `Ctrl+R expand`, `Shift+Tab modes`, `Ctrl+_ undo`, `↑ history`, `?? help`.

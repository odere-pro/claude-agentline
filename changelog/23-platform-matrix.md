Prove every release works on every supported host before it ships: the install-matrix workflow now packs the repo with `npm pack` on macos-13/14, ubuntu-22.04/24.04, and windows-2022 × Node 20/22, installs the resulting tarball into an isolated `--prefix`, pipes a canonical Claude Code statusline payload through the freshly installed bin, and asserts the exact stdout — so a publish-time regression in packaging, bin resolution, or the render path is caught on every PR. The same recipe lands locally as `tests/gates/gate-15-platform-matrix.sh` so contributors can validate before pushing. The workflow then runs `install.sh && uninstall.sh` against a temp `HOME` / `CLAUDE_CONFIG_DIR` to keep gates 07/09/10 honest, and uploads `.ci-pack/*.tgz` plus `tests/gates/.tmp/` as failure artefacts for triage.

- Editor preview wrap is now deterministic instead of relying on Yoga's `flexWrap`: the outer preview `Box` has no defined width, so Yoga was happy to let row content grow past the terminal edge — what looked like a wrap on narrow terminals was actually the terminal's own column overflow. The preview now measures each slot against `process.stdout.columns`, packs slots into visual sub-lines, and indents wrap continuations so they line up with the gutter (a screenshot with eleven row-0 widgets used to render every cell on one ragged line; the trailing cells now wrap whole and stay aligned).

- The TUI editor's per-widget options sheet is retired. The `o` keybinding, the dedicated `options` scope, and the `OptionsSheet` overlay component are all gone — the `v`/`l`/`m` toggles inside the sheet are now set via `agentline config widget set-option <key> <value>` (or by hand-editing the config file). The user-facing motivation: the sheet was an extra overlay layer that duplicated CLI surface area, broke the picker's "open / close / continue" rhythm, and made the editor harder to teach. With the sheet gone the editor has exactly two modes (`edit` and `picker`) and `↵` on a populated widget is a no-op (the surfacing paths users actually need — change variant, replace, delete — are still on `u` / `r` / `d`). Plumbing across the deletion: `src/tui/options-sheet.ts` and its test file are deleted; the `o` row and the four `options`-scope rows (`v`/`l`/`m`/`Esc`) come out of `DEFAULT_KEY_BINDINGS`; `KeyScope` drops `"options"`; `EditorMode` drops `"options"`; the reducer drops the `open-options` / `close-options` / `toggle-hidden` / `toggle-raw` / `cycle-merge` actions and the dead `MERGE_CYCLE` / `nextMerge` helper they were the only callers of (the `set-option` action stays — that's what the new `agentline config widget set-option` CLI path drives through the reducer when callers want to). The footer's `FOOTER_LABEL` map drops the five labels for the removed actions; `modeToScope` / `SCOPE_HEADING` / `SCOPE_ORDER` lose their `options` branch. gate-17-keymap-coverage and `bindings.test.ts` drop the same five actions from their required-action list, in lockstep. `docs/keymap.md` and SPEC §5.5 are rewritten to describe a two-mode editor and point users at the CLI for the per-widget toggles. State tests are simplified — the toggle / cycle suite is replaced by a single `mark-clean` regression and the `set-option` suite is preserved (still validating the prototype-pollution + empty-key rejects and the multi-call merge). 1123 passing across 112 files (was 1130 on PR #74; the net `-7` is options-sheet test scaffolding); 12 gates green.

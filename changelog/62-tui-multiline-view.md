- Rework the `agentline config` editor view for multi-line editing and the new keymap. The layout view now shows every configured row (up to three), with the selected widget highlighted; `←/→` move the selection within a row, `↑/↓` move it to the adjacent row, `⇧+arrows` carry the selected widget (within a row or to the next, creating a row when there's headroom). `a` adds / `r` replaces a widget (a small curated rotation for now — the picker overlay lands separately), `x` deletes, `v` toggles visibility, `m` cycles spacing-to-neighbour, `l` toggles the widget's own label, `?` toggles a help overlay listing the keys. The keymap registry (`src/keys/bindings.ts`), spec §5.5, and gate-17's action list are rewritten in lockstep to this set; `agentline config keys [--json]` reflects it. Reducer behaviour is unchanged (the new view dispatches the reducer actions added last PR; the legacy single-line actions remain for now). The picker overlay (`a`/`r` → an autocomplete chooser using the demo previews) and the options sheet (`o`) are still to come.

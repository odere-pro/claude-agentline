- `agentline config` gains a per-widget options sheet. Pressing `o` on a selected widget opens an overlay that spells out, in plain English, the three commonly-toggled properties — visible / hidden, the widget's own label shown / hidden, and spacing to the neighbour (full padding / single space / none) — with the current value of each; `v` / `l` toggle, `m` cycles spacing, `Esc` closes. New `src/tui/options-sheet.ts` (the `OptionsSheet` Ink component plus a pure `optionsSummary` helper); the same toggles still work directly from the layout view. 5 new tests. (The `o` / `Esc` and `picker` keys are shown in their overlays; folding them into `agentline config keys` / spec §5.5 is the remaining keymap tidy-up.)

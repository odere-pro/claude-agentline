- The CLI surface is now flat. The `--preset` and `--target` flags are gone, the `minimal` / `maximal` templates are deleted, and only the `claude-code-dark` theme is shipped (the other three themes are gone; users who want them can drop JSON into their themes dir). `src/cli.ts` now exports a `COMMANDS` dispatch table and a new `src/cli.test.ts` asserts the exact surface (`render` / `edit` / `init` / `keys` / `install` / `uninstall` / `doctor` plus the `help` / `version` aliases) so a stray re-add can't slip past review; auto-run is gated on a realpath comparison so the npm-installed bin (which reaches `dist/cli.mjs` through a symlink) still triggers `main()`. Spec §9.1 is rewritten to match. `theme/listThemesIn` is dropped (its only caller was the deleted theme command).
